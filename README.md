<p>
In this project we will be using Azure to simulate the implementation of active directory onto a Windows server VM which can also be called the Domain Controller. The first step in this project is the creation of the Windows VMs. In Azure create a Windows Server VM and a normal Windows 10 VM. Keep track of the username and password of both VMs in a note document. Also make sure that for the Windows 10 VM you select the same resource group and virtual network of the domain controller because this lab wouldn't work without them being on the same virtual network. Next in Azure go to the Domain Controller VM and click on the networking tab, then click on the network interface. Then go to the IP config setting click on the bar containing the private and public IP adresses. Then set the assignment to static. This makes the IP of the VM static.
